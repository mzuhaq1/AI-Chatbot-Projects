This repository contains multiple chatbot implementations built using Ollama and Python, focusing on lightweight, local AI deployment.

- Uses TinyLlama model
- Command-based interaction

### 2. Python-Based Chatbot

- Integrated with Ollama API
- Interactive chatbot loop
